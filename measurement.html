1. Selectors - Selectors are patterns used to select the elements you want to style.

  • Element Selector
  • Class Selector
  • ID Selector
  • Attribute Selector
  • Pseudo-class & Pseudo-element

2. Properties - Properties define what aspect of the selected element you want to style.

  • Color
  • Font
  • Background
  • Border
  • Margin & Padding

3. Values & Units - Values are assigned to properties, often using specific units.

  • px, em, rem, %, vw, vh
  • Color values (hex, rgb, hsl)
  • Keywords (auto, inherit, initial)

4. Box Model - The box model describes how elements are rendered and spaced.

  • Content
  • Padding
  • Border
  • Margin

5. Layout & Positioning - CSS provides ways to control the layout and position of elements.

  • Display
  • Position (static, relative, absolute, fixed, sticky)
  • Flexbox
  • Grid
  • Float & Clear
